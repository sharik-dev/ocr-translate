# meowToon

A SwiftUI iOS app built by Sharik Mohamed.

## Requirements

- Xcode 15+
- iOS 17+
- Swift 5.9+

## Getting Started

1. Clone the repo:
   ```bash
   git clone <your-repo-url>
   ```
2. Open `meowToon/meowToon.xcodeproj` in Xcode.
3. Select a simulator or device and press **Run** (⌘R).

## Project Structure

```
meowToon/

│   ├── meowToonApp.swift
│   └── ContentView.swift
├── meowToonTests/      # Unit tests
└── meowToonUITests/    # UI tests
```
